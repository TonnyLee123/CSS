# 3. Padding 
- Create space around an element's content, inside of any defined borders.

```css
div {
  padding-top: 25px;
  padding-right: 50px;
  padding-bottom: 75px;
  padding-left: 100px;
}
----------上右下左-----------
div {
  padding: 25px 50px 75px 100px;
}
上右下左
-----------全一樣------------
div {
  padding: 25px;
}
```
